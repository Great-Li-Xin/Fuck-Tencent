# fuckLOL
> LOL and OW is the most lowest game in the world

### 背景：
作为一个学习党，总是被室友的打游戏的机械键盘声音吵到么，你还总是被室友的室友要被杀死的绝望叫声所震撼么，你还被室友输了游戏以后久久不能释怀的喃喃自语所心烦么。不用心烦，不用慌张。室友团战断网必备。

### 设计原则
1. 绝对不打扰到非游戏玩家的网络访问。
2. 绝对不由抢网速的原因来进行局域网攻击。

### 支持平台
1. OS X (就是不叫Mac OS)
2. Linux

### 所用库/所需要的环境
1. scapy 绝对强大的linux/unix开源网络库
2. todo

### 支持断网游戏
1. LOL
2. OW
